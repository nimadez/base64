# Base64 Editor

Base64 encoder, decoder, image editor and font viewer<br>
**1.3.0 Alpha 2022**

## Features
- Base64 encoder and decoder
- Support text, file, image or font
- Support unicode characters (emoji, asian and middle-eastern languages)
- Base64 image preview and resizer (pixel and percent)
- Base64 font preview
- Download content as text, encoded text or data (see supported formats)
- Copy/Paste clipboard
- Fluid user-interface for desktop and mobile
    
## Supported Data Formats
- Image: JPG, PNG, GIF, SVG, ICO, TIFF
- Font: TTF, WOFF, WOFF2
- Audio: MP3, WAV, WEBA
- Video: MP4, WEBM
- Data: HTML, JSON, JS, CSS, XML
